ĐÂY LÀ HƯỚNG DẪN SỬ DỤNG GITHUB

Check mình đang ở nhánh nào
git checkout 

Ví dụ ở đây có 4 nhánh 
    + Hai
    + bach
    + Bao
    + Mai
thì khi cập nhật code sẽ chuyển nhánh đó
git checkout <tên_nhánh>

cập nhật code của mọi người về nhánh mình
git pull origin <tên_nhánh>
